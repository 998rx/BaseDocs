Getting Started

VuePress is a Markdown-centric static site generator. You can use Markdown to write content (like documentation, blogs, etc.), and VuePress will help you generate a static site to showcase them.

How it works?

A VuePress site is essentially a Single-Page Application (SPA) powered by Vue and Vue Router.

Routes are automatically generated based on the relative paths of your Markdown files. Each Markdown file is compiled into HTML via markdown-it and then used as a Vue component template.

Features

  • Built-in Markdown extensions
  • Vue-powered custom theme system
  • Plugin API
  • Default theme optimized for technical documentation

Quick Start

npm install vuepress@next

Create your first document:

# Hello VuePress

Welcome to your VuePress site!
